#CUCUMBER-TESTING
## Proyecto de automatización
En este proyecto se automatiza con el framework:
- Cucumber
- Testng
- Selenium
- Allure Report

## Configuración para pruebas Testng
Ir al archivo testng.xml activar las configuraciones

## Ejecución de pruebas
```bash
  mvn clean test
```

## Ejecución Allure report
```bash
  mvn allure:serve
```
